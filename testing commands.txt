to compile with gcc (MinGW):
gcc node.c -o node.exe -lws2_32

to compile with clang:
clang node.c -o node.exe -lws2_32

=== SIMULATION MODE (event file tests) ===

:: Algorithm comparison (run all 3 to see differences)
node A.conf tahoe events/comparison_test.txt
node A.conf reno events/comparison_test.txt
node A.conf newreno events/comparison_test.txt

:: NewReno partial ACK advantage
node A.conf newreno events/newreno_partial_ack_test.txt
node A.conf reno events/newreno_partial_ack_test.txt

:: Long session with multiple losses
node A.conf newreno events/long_session_test.txt
node A.conf tahoe events/long_session_test.txt

:: Original test files
node A.conf newreno events/example_events.txt
node A.conf newreno events/duplicate_ack_test.txt
node A.conf newreno events/timeout_test.txt

:: Different nodes with different configs
node B.conf newreno events/comparison_test.txt
node C.conf newreno events/comparison_test.txt
node D.conf newreno events/comparison_test.txt

=== INTERACTIVE NETWORK MODE (6 terminals) ===

:: Open 6 separate terminals and run:
:: Terminal 1:
node A.conf newreno
:: Terminal 2:
node B.conf newreno
:: Terminal 3:
node C.conf newreno
:: Terminal 4:
node D.conf newreno
:: Terminal 5:
node E.conf newreno
:: Terminal 6:
node F.conf newreno

:: Then from Node A terminal:
:: > send D hello_from_A_to_D
:: Expected: A->B->D forwarding

:: From Node F terminal:
:: > send E hello_from_F_to_E
:: Expected: F->A->B->E forwarding

:: To run simulation inside interactive mode:
:: > run events/comparison_test.txt

:: To view routing table:
:: > table

:: To exit:
:: > quit
